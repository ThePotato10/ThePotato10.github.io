Timer for Workouts

Sometimes when you're working out, you need to immediately transfer from doing something for 30 seconds to doing something for 45 seconds. Stopping to reset a timer defeats the purpose of the set, and counting in your head is innaccurate. This website allows you to schedule several intervals in a row, fixing this problem. Simply type in the exercise you're doing, how long you want to do it for, and how long you want to rest afterwards. If you don't want to rest afterwards, simply type '0' in the Rest Duration box. When you hear a ding, that's the end of the current interval. Schedule as many intervals as you would like!
